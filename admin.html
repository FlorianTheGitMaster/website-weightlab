Anmelden

Zum Bearbeiten brauchst du ein GitHub-Token mit Schreibzugriff auf FlorianTheGitMaster/website-weightlab (Contents: Read & Write). Das Token bleibt nur in diesem Browser (localStorage) — es wird nirgendwo sonst hingeschickt außer an die GitHub-API.



Welche Website ist live?

Der Umschalter wirkt auf index.html — Besucher der Domain sehen die gewählte Website. (Nach dem Speichern dauert es je nach Hosting ca. 1–2 Minuten.)

Akzentfarbe der Post-Website

Weiß bleibt Basis — die Akzentfarbe ist frei wählbar (Salbei, Schwarz, …).

Bilder der Post-Website

Bilder landen als Commit im Ordner img/. Quer fürs Web, ideal < 500 KB pro Bild (vorher verkleinern).
